# Loja-e-Fjal-ve
Loja e fjalëve e bazuar në Java-Swing

Sigurohuni qe e keni java te instaluar ne kompjuter dhe kompajlloni projektin per ta startuar...

Rezultatet e fundit jane te ruajtura tek file i quajtur rezultati.txt!
Fjalet e lojes jane te ruajtura tek file i quajtur fjalët.txt dhe ju mund ti ndryshoni ne fjalë të reja!
